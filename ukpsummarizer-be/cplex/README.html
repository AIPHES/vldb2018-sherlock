IBM ILOG CPLEX Optimization Studio Version 12.8.0

IBM® ILOG CPLEX Optimization Studio ist die schnellste Methode für das Erstellen effizienter Optimierungsmodelle und technologisch ausgereifter Anwendungen für den gesamten Bereich von Planungs- und Zeitplanungsproblemen. Mit seiner integrierten Entwicklungsumgebung (IDE, Integrated Development Environment), seiner beschreibenden Modellierungssprache und seinen integrierten Tools sowie technologisch ausgereiften Solvern für Probleme bei der mathematischen Programmierung und der Constraintprogrammierung unterstützt das Produkt den gesamten Modellentwicklungs- und Lösungsprozess.

CPLEX Optimization Studio setzt sich aus mehreren Komponenten zusammen. Nicht alle Komponenten sind auf allen Plattformen verfügbar. Weitere Einzelheiten hierzu finden Sie auf der Webseite mit den detaillierten Systemvoraussetzungen. Jede Komponente befindet sich in einem Unterordner (in einem Unterverzeichnis unter AIX und Linux) der Installationsposition. Die Anwendungsprogrammierschnittstellen (API, Application Programming Interface) und Connectors für die einzelnen Komponenten befinden sich im Unterordner (Unterverzeichnis) der jeweiligen Komponente. Die MATLAB- und AMPL-Connectors für CPLEX befinden sich beispielsweise im Ordner (Verzeichnis) cplex.

  • opl: OPL-Modellierungsumgebung und integrierte Entwicklungsumgebung (IDE, Integrated Development Environment)
  • cplex: CPLEX Optimizer (u. a. Simplex Optimizer, Barrier Optimizer und MIP Optimizer)
  • cpoptimizer: CP Optimizer mit Constraintprogrammierungstechnologie
  • concert: C++-Modellierungsschicht für die Optimizer
  • python: Python-Modellierungsschicht DOcplex für CPLEX Optimizer und CP Optimizer

Weitere Verzeichnisse:

  • doc: HTML-Version der Dokumentation
  • jre: Java-Laufzeitumgebung für den Installations- und Deinstallationsprozess (auf Plattformen ohne OPL IDE)
  • license: Lizenzvereinbarungsdokumente
  • properties: Inventarinformationen
  • Uninstall: Für die Deinstallation des Produkts erforderliche Informationen

Installation unter Windows

Unter Windows ist die Standardinstallationsposition von IBM ILOG CPLEX Optimization Studio folgende:
C:\Program Files\IBM\ILOG\CPLEX_Studio128

Installationsposition für die Community Edition:
C:\Program Files\IBM\ILOG\CPLEX_Studio_Community128

Möglicherweise haben Sie während der Installation eine andere Position ([Installationsverzeichnis]) ausgewählt.

IDE

Sie können die integrierte Entwicklungsumgebung (IDE) über einen Eintrag im Menü "Start" starten:

Start > Alle Programme > CPLEX Optimization Studio 12.8.0 > CPLEX Studio IDE

Community Edition:

Start > Alle Programme > CPLEX Optimization Studio Community Edition 12.8.0 > CPLEX Studio IDE

Weitere Informationen zum CP Optimizer auf Windows-Plattformen finden Sie in der Datei [Installationsverzeichnis]\cpoptimizer\readmeWindows.html.

Installation unter AIX und Linux

Die Installationsprogramme für AIX und Linux sind ausführbare Dateien. Zum Starten einer Installation benötigen Sie Ausführungsberechtigung für diese Dateien, die Sie über die Befehlszeile starten müssen. Setzen Sie beispielsweise den Befehl ./<Installationsprogramm>.bin ab. Hier steht <Installationsprogramm> für den Namen des Installationsprogramms. Mit dem Befehl chmod u+x <Installationsprogramm>.bin können Sie die Ausführungsberechtigung für die Datei erteilen.

Unter AIX ist die Standardinstallationsposition von IBM ILOG CPLEX Optimization Studio folgende:
/opt/IBM/ILOG/CPLEX_Studio[Edition]128

Unter Linux ist die Standardinstallationsposition von IBM ILOG CPLEX Optimization Studio folgende:
/opt/ibm/ILOG/CPLEX_Studio[Edition]128

[Edition] kann folgende Werte annehmen:
* Community für die Community Edition
* leer für die kommerzielle Standardedition

Möglicherweise haben Sie während der Installation eine andere Position ([Installationsverzeichnis]) ausgewählt.

IDE

Sie können die IDE von einem Linux-Terminal starten.
1. Öffnen Sie ein Terminalfenster.
2. Rufen Sie das Verzeichnis [Installationsverzeichnis]/opl/oplide auf.
3. Geben Sie den Befehl ./oplide ein.

Geben Sie alternativ an einer beliebigen Terminalposition den absoluten Pfad ein:
[Installationsverzeichnis]/opl/oplide/oplide
